
# Untitled: Notes & Reminder 
Notes and Reminder taking app, MVVM with Google Architectural components Room, LiveData and ViewModel written in Java, androidx libraries


## Features

- Simple and Quick
- Add Reminders
- Supported Light/dark mode
- Fullscreen mode
- Re-edit Notes
